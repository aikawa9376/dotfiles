# Cursor Context

Get information about the current state of the Neovim editor.

## Usage Scenarios
- **"カーソルを合わせて"**: Understand where the user is currently looking.
- **"今の場所を確認して"**: Get the surrounding code context of the cursor.
- **"Where am I?"**: Find out which file and line the editor is on.

## Commands

### cursor
Get the current cursor position, file path, and surrounding code context.
```bash
nvim-cli cursor
```
